### Installation

 - Clone this project
 - Install module dependencies by run this script in terminal
    `npm install`
 - Finally run the app.
	 `npm run dev`
 - Navigate to  [http://localhost:3000](http://localhost:3000)

### Deployment

 - First you need to build the production assets first
    `npm run build`
 - Then start the app
    `npm start`
 - Navigate to  [http://localhost:3000](http://localhost:3000)
